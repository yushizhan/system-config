# 新环境的一些配置

## Vim配置
将仓库clone到本地文件夹.vim，然后修改.vimrc：

```
source ~/.vim/.vimrc
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
打开Vim，运行：PluginInstall
```
